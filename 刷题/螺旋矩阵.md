## 螺旋矩阵 II

- 题目链接：https://leetcode-cn.com/problems/spiral-matrix-ii/
- Tag：数组

## 题目描述
给定一个正整数 n，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的正方形矩阵。

示例:

输入: 3
输出:
```
[
 [ 1, 2, 3 ],
 [ 8, 9, 4 ],
 [ 7, 6, 5 ]
]
```

## 解法

```
class Solution {
    public int[][] generateMatrix(int n) {
        int[][] rt = new int[n][n];
        int count = 1;
        int x = 0, y = 0;
        while(count <= n * n) {
            while(y < n && rt[x][y] == 0) {
                rt[x][y] = count;
                count++;
                y++;
            }
            y--;
            x++;
            while(x < n && rt[x][y] == 0) {
                rt[x][y] = count;
                count++;
                x++;
            }
            x--;
            y--;
            while(y >= 0 && rt[x][y] == 0) {
                rt[x][y] = count;
                count++;
                y--;
            }
            y++;
            x--;
            while(x >= 0 && rt[x][y] == 0) {
                rt[x][y] = count;
                count++;
                x--;
            }
            x++;
            y++;
        }
        return rt;
    }
}
```
