## 合并排序的数组

## 题目描述
给定两个排序后的数组 A 和 B，其中 A 的末端有足够的缓冲空间容纳 B。 编写一个方法，将 B 合并入 A 并排序。

初始化 A 和 B 的元素数量分别为 m 和 n。
```
示例:

输入:
A = [1,2,3,0,0,0], m = 3
B = [2,5,6],       n = 3

输出: [1,2,2,3,5,6]
说明:

A.length == n + m
```
## 解法
将A数组向后挪动n个位置，然后两个指针分别指向A、B中第一个元素位置，比较大小顺序放入A中。
```
class Solution {
    public void merge(int[] A, int m, int[] B, int n) {
        for(int i = m - 1; i >= 0; i--) {
            A[n + i] = A[i];
        }
        int x = n, y = 0;
        int index = 0;
        while(x < m + n && y < n) {
            if(A[x] <= B[y]) {
                A[index] = A[x];
                x++;
                index++;
            }else {
                A[index] = B[y];
                y++;
                index++;
            }
        }
        while(x < m + n)  {
            A[index] = A[x];
            x++;
            index++;
        }
        while( y < n) {
            A[index] = B[y];
            y++;
            index++;
        }
    }
}
```
